    微软拼音输入法介绍

    本输入法是一个汉语拼音语句输入法，用户可以连续输入汉语语句的拼音，系统会自动选出拼音所对应的最可能的汉字，免去逐字逐词进行同音选择的麻烦。

    微软拼音输入法的另外一个特点是支持音调，在拼音后面加上音调可以减少同音字的选择。

    激活中文输入法

    按住 CTRL 键不放，然后按空格键，可以激活中文输入法，此时在屏幕左下方出现输入法状态条。

    在中文输入法状态下，按住 CTRL 键不放，然后按空格键，将取消中文输入法，回到英文输入状态。

      输入法状态条

      输入法状态条表示当前的输入状态，单击上面的按钮可以切换输入状态，拖动输入法状态条左端，可以将它移到屏幕的任何地方。根据不同厂商的配置，微软拼音输入法会提供两种形式的状态条。

      输入法状态条按钮的含义分别是：

      • 中文/英文切换按钮： 表示中文输入， 表示英文输入。
      • 全角/半角切换按钮： 表示全角符号， 表示半角符号。
      • 中/英文标点切换按钮： 表示中文标点， 表示英文标点。
      • 输入法设置按钮： 点击它可以设置输入法的某些选项。

        中文输入
        • 标识图符：。
        • 鼠标操作：鼠标左键单击输入法状态条的中英文切换按钮使出现标识图符。
        • 快捷键： Shift。

        中文微软拼音输入法支持两种拼音输入方式：全拼输入和双拼输入。并且在两种输入方式中都可以支持带调、不带调或二者的混合输入。 输入法分别以数字键1,2,3,4代表拼音的四声，5代表轻声。输入的各汉字拼音之间无需用空格隔开，输入法将自动切分相邻汉字的拼音。当然，对于有些音节歧义，目前系统还不能完全识别，此时需用音节切分符{Space}来断开（详见音节切分符）。另外, 带音调拼音输入的自动转换准确率将高于不带音调的拼音输入。

        双拼输入

        为了提高输入速度，系统支持双拼输入。依据双拼输入键位表，每个汉字对应两键，第一键为声母，第二键为韵母。

        汉语中有些音节只有韵母，没有声母，称为零声母音节。在输入零声母字时，先输入一个字母"ｏ"，之后连续输入相应的韵母便可完成一个零声母汉字的输入。虽然输入的为双拼，拼音窗口中显示的仍为该音节的全拼形式，只是在显示该拼音时，我们在该字的拼音之前加"`"代表输入的是零声母字。例如，用户输入"oa"，在拼音窗口中显示"`a"，系统将给出拼音为"a"的汉字。

        • 用户可以在输入法设置对话框中设置双拼输入方法。
        • 操作举例
          用户希望输入"这是一只可爱的小花猫"，可以带音调输入
          "ve4ui4yi1vi1ke3ol4de5xc3hw1mk1{Space}"，
          也可以不带音调输入
          "veuiyivikeoldexchwmk{Space}"，
          也可二者混合输入
          "ve4ui4yi1vikeoldexchwmk{Space}"。

        音节切分符

        微软拼音使用空格" "或单引号"'"作为音节切分符。也可以键入音调来代替音节切分符。

        对于一些拼音词组，目前输入法还难以自动切分歧义音节，如"xian"，用户希望得到的是"西安"，而输入法可能转换为单字"先"。解决的方法是用户可以使用音节切分符。在"xi"和"an"之间键入一个空格（"xi an"）或在"xi"后面键入"xi"字的音调"1"("xi1an"），来避免这类错误。

        确认输入

        在您确认输入之前，输入窗口中的内容并没有传递给编辑器，这时如果您按了ESC键，输入窗口中的内容将全部丢失。 要完成输入过程，您可以按回车或者空格键。

        • 按回车键，输入窗口中的内容，包括未转成汉字的拼音，将全部传递给编辑器。
        • 按空格键，如果输入窗口中还有未转成汉字的拼音，则先转成汉字。

        此外，当您敲入逗号、句号、分号、问好和感叹号后，输入法在下一句的第一个声母键入时，自动确认该标点符号之前的句子。

        错字修改

        当用户连续输入一串汉语拼音时，微软拼音输入法通过语句的上下文自动选取最优的输出结果。但在一些情况下，输入法自动转换的结果与用户希望的有所不同时，用户可以通过输入法提供的候选字/词功能修改输入法的输出结果。另外，本输入法是以句子为单位的输入法，所以它有一些使用技巧。

        操作步骤：

        1. 用鼠标或键盘移动光标到错字处，键盘的定义可以参考操作键定义表。
        2. 候选窗口自动打开。
        3. 用鼠标或键盘从候选窗口中选出正确的字或词。

        拼音错误修改

        用户可以修改已转换汉字的拼音。当输入的中文语句还未确认以前，用户可以用键盘上的 → 或 ← 键移动光标到拼音有误的汉字前，按下"`"键(在"Tab"键上方)，输入法弹出拼音窗口，用户可以在此窗口中重新键入汉字的正确拼音。注意，只有在候选窗口激活的情况下，按"`"才做激活拼音窗口之用，否则，将直接插入字符"`"。如果候选窗口没有弹出，在待修改字前按空格键激活候选窗口。

        错字修改的技巧
        • 微软拼音输入法的基本输入单位为语句，这是它区别于其他输入法的显著特点，因此在输入语句时，发现有错别字不用忙于修正，最好是在确认语句之前对整句一起修改。在输入的过程中，微软拼音输入法会自动根据上下文做出调整，将语句修改为它认为最可能的形式。往往经过它的调整，很多错误就会自动消失。
        • 修改句子最好从句首开始。
        • 输入完一个句子，按右方向键可以快速回到句首。光标移动键的作用是循环的。

        零声母与音节切分符

        汉语拼音中有一些零声母字，即没有声母的字，例如"奥"（ao4），"欧"（ou1）等。在语句中输入这些零声母字时，使用音节切分符可以得到事半功倍的效果。例如，输入"皮袄"时，输入带音节切分符的拼音"pi ao"（中间加一个空格），可以省去很多修改的麻烦。

        汉语拼音标准

        微软拼音输入法采用的汉字标音方案遵照1958年2月11日第一届全国人民代表大会第五次会议批准的《汉语拼音方案》。为了在键盘上的实现方便，微软拼音输入法用字母v表示ü。此外，《汉语拼音方案》中一些特殊发音汉字的标注在微软拼音输入法中也采用了不同的记法。微软拼音输入法与《汉语拼音方案》的不同处如下表所示：

        汉语拼音方案微软拼音输入法
        üv
        诶 êea
        hmhen
        哼 hngheng
        呒 mmu
        嗯 n, ngen

        用户自造词

        在您使用输入法的过程中，随时可以使用自造词功能。只要在输入窗口中块选您要定义的词，然后按回车即可。

        用户自造词必须由2至9个汉字组成，不能有英文字母和标点符号。

        自学习

        微软拼音输入法具有的自学习功能会记住用户在组字窗口中所作的修改，并调整相应字词的出现频率，使用户在下一次输入同样的词语时，提高转换准确率。
