      About connecting to the Internet

        You can use your mobile device to connect to the Internet, which enables you to browse the World Wide Web (WWW) and exchange e-mail.

          If your device is not equipped with a built-in modem, you will need to install a modem card or external modem. See your modem manufacturer's instructions for details on installation. Once the modem is set up, create a connection to an ISP.

            See Also
            Connect to the Internet using an ISP
            Create an ISP account

            Create an ISP account

              Note PPTP accounts are not supported on your device.

                Set up a Point-to-Point Protocol (PPP) account with an Internet Service Provider and obtain the following information:

                • Access telephone number
                • User name
                • Password

                After you establish an account, create a new connection on your device. When creating this connection, you should be able to use all of the default TCP/IP settings provided in the Make New Connection Wizard. If you can't connect using the default settings, contact your ISP for specific TCP/IP information, and for primary and secondary DNS addresses.

                  Create a new connection on your device

                  Create a new connection on your device
                    1. Select Start > Settings > Network and Dial-up Connections.
                    2. Double-tap or double-click the Make New Connection icon.
                    3. In the Make New Connection dialog box, enter a name for the connection.
                    4. Select Dial-Up Connection.
                    5. Select the Next button.
                    6. Select the modem you want to use from the drop-down list.
                    7. If you want to enter a new phone number each time you connect, select Configure.
                      • In the Device Properties dialog box, Select the Port Settings tab.
                      • Check Manual Dial to be able to enter a phone number every time you connect.
                      • If you want to customize telephone options, select the Call Options tab and set option preferences.
                      • Select OK to accept your configuration choices.
                    8. Select TCP/IP Settings.
                      • In the General tab, ensure Use Server-assigned IP address is checked.
                      • In the Name Servers tab, ensure Use Server-assigned addresses is checked.
                      • Select OK.
                    9. Select the Next button.
                    10. In the Phone Number dialog box, enter the telephone access number for your ISP.
                    11. Select the Finish button.

                    The connection you just created appears as an icon in the Network Connections panel.

                    Notes

                    • If you are unable to connect with the default settings, see your ISP or network administrator for assistance.
                    • If you are creating a PC modem connection for an external modem, select Hayes Compatible on COM1 as your modem.
                    • When you connect your device to a PC using an external modem, use the modem cable or adapter provided by the manufacturer of your modem.

                        About connecting to a network

                          You can connect directly to a network through a network (Ethernet) or dial-up connection. You can access your e-mail, access files available on the network server, and browse the Internet or an intranet. To receive e-mail, once you have created the connection to the network, create either a POP3 or IMAP4 mail service in Inbox. For more information, see Inbox Help.

                            See Also
                            Create a network (Ethernet) connection
                            Create a dial-up network connection

                            Create a network (Ethernet) connection

                              1. On your device, insert the Ethernet card.
                              2. In the Ethernet Driver Settings dialog box, select Obtain an IP address via DHCP. Select OK.
                              3. Select Start > Settings > Control Panel. Double-tap or double-click the Network and Dial-up Connections icon.
                              4. Double-click the connection icon for the adapter, and click OK to verify settings.
                                For example, if you have a DC21x4 Ethernet adapter, double-click the DC211401 connection icon .
                              5. When prompted, enter the User Name, Password, and Domain name you use to log on to your network.

                                See Also
                                Connecting to a Network

                                Create a dial-up network connection

                                1. Ask your network administrator to create a RAS account for you. Once the account is set up, obtain the following information: dial-up phone number, password, and computer name.
                                2. Make a new connection on your device using the information in step 1. You should be able to use the default TCP/IP settings provided in the Make New Connection Wizard. If you can't connect using these default settings, contact your network administrator for specific TCP/IP information.

                                Create a new connection on your device

                                Create a new connection on your device

                                  You must create a connection for each computer or network that you want to access remotely unless you are using an Ethernet card.

                                  1. Select Start > Settings > Network and Dial-up Connections.
                                  2. Double-tap or double-click the Make New Connection icon.
                                  3. In the Make New Connection dialog box, enter a name for the connection.
                                  4. Select Dial-Up Connection and click the Next button.
                                  5. Select the modem you want to use.
                                  6. If you want to enter a phone number every time you connect,
                                    • Select Manual Dial, then click OK.
                                    • Select TCP/IP Settings.
                                    • In the General tab, ensure Use Server-assigned IP address is checked.
                                    • In the Name Servers tab, ensure Use Server-assigned addresses is checked, and select OK.
                                    • Select the Next button and type the telephone number.
                                    • Select the Finish button.
                                  The connection you just created appears as an icon in the Remote Networking folder.
                                    Notes
                                    • If you are unable to connect with these default settings, see your ISP or network administrator for specific configurations or TCP/IP information.
                                    • The connection appears as an icon on the Connections screen.
                                    • If you are creating a PC modem connection for an external modem, select Hayes Compatible on COM1 as your modem. When connecting your device to your PC using an external modem, use the modem cable or adapter provided by the manufacturer of your modem.

                                    Connect to a network

                                    If you connect to a network, you will be able to receive e-mail and browse the Internet only. To receive e-mail, create either a POP3 or IMAP4 mail service in Inbox. For more information, see Inbox Help.

                                      1. Select Start > Settings > Network and Dial-up Connections.
                                      2. Double-tap or double-click the remote connection icon you want to use.

                                        The first time you connect using the connection, you will need to enter your user name, password and domain. To prevent having to re-enter your password on subsequent connections, select Save password.
                                      3. Select Connect.
                                      4. To disconnect, select Disconnect.

                                      Connect to a wireless network
                                      1. Plug in your wireless card. If prompted, supply the name of the driver provided by the manufacturer.
                                      2. Once loaded, the system will pop up a dialog to assist you in connecting to available wireless networks.
                                      3. Select the name of the network that you want to connect to, and click Connect.
                                      4. Confirm the settings that were detected, and click OK.
                                      5. If you need to review or change settings, select Start > Settings > Network and Dial-up connections.
                                      6. Double-click the remote connection icon that you want to use.
                                      7. If the driver for your wireless card supports it, a Wireless Networks tab will be available to adjust settings.
                                      8. Use this tab to view, change, or adjust settings.

                                      Adjusting Remote Connection Properties

                                        Modify connection settings
                                        Turn software compression off
                                        Change the baud rate for your device
                                        Modify connection settings
                                        1. Select Start > Settings > Network and Dial-up Connections.
                                        2. Select the icon for the connection settings you want to modify.
                                        3. Select File > Properties, or double-click the appropriate icon.
                                        4. Select desired options. There may be additional settings that depend on the connection. To modify, select the icon and select Advanced Settings.. from the menu.
                                        See Also
                                        Create a new connection on your device
                                        Turn software compression off
                                        1. Select Start > Settings > Network and Dial-up Connections.
                                        2. Select the icon for the connection whose settings you want to modify.
                                        3. Select File > Properties.
                                        4. Select TCP/IP Settings.
                                        5. In the General tab, clear Use software compression.

                                        Note While turning off software compression will shorten the time it takes to complete a connection, it slows the rate of data transmission. Only turn off software compression if you connect frequently and transfer small amounts of data in each session.

                                          Change the baud rate for your device

                                            Important Changing this setting may disable communications with your PC. The baud rate you selected for your device must match the baud rate on your PC. For more information on changing the baud rate on your PC, see Windows CE Services Help.

                                            1. Select Start > Settings > Control Panel.
                                            2. Double-tap or double-click the PC Connections icon.
                                            3. Select Change ....
                                            4. Select the desired baud rate.

                                              About Terminal

                                                You can connect your device to an online service or corporate server that requires TTY or VT-100 terminal emulation. For example, you can view and download files from a bulletin board or send and receive e-mail.

                                                  Use Terminal only when communicating with services that require terminal emulation.

                                                    See Also
                                                    Connecting to a Network
                                                    Connecting to an ISP

                                                    Create a new Terminal session
                                                    1. Select Start > Settings > Control Panel > Terminal Server Client Licenses.
                                                    2. Double-tap or double-click the Make a New Session icon.
                                                    3. In the Session Name box, enter a name for the session.
                                                    4. In the Select a Modem list, select the name of your modem.
                                                    5. Enter the telephone number for the remote computer.
                                                    6. Unless you need to adjust the emulation properties, select OK to connect.
                                                    Notes
                                                    • The session you created appears as an icon in the Terminal folder.
                                                    • To disconnect, select File > Cancel.
                                                    See Also
                                                    Adjust session properties
                                                    Connect using an existing session
                                                    Connect using an existing session
                                                    • In the Terminal window, double-tap or double-click the icon for the session you want to use.
                                                      When your device has connected to the online service, the Terminal window appears.
                                                    • You can create a desktop shortcut for the session, and connect by double-tapping or double-clicking it.
                                                    • To disconnect, select File > Cancel.
                                                    See Also
                                                    Create a new Terminal session
                                                    Adjust session properties

                                                    Adjust session properties
                                                    1. In the Terminal window, select the icon for the session you want to modify.
                                                    2. Select the Properties button.
                                                    3. In the Communications tab, modify the settings as needed.
                                                    4. To modify the emulation settings, select the Emulation tab.
                                                    5. In the Choose an emulation type list, select the type of terminal to emulate.
                                                    6. In the Code page selection list, select the code page option for the character set you want.
                                                    7. To display text you type before sending, select Local Echo.
                                                    8. In the Use small font by default box, set your font preference.
                                                    9. In the CR -> CR/LF box, adjust the carriage return/line feed, and select Inbound or Outbound settings.
                                                    10. In the Automatic Scrolling box, set your scrolling preference.
                                                    11. When finished, select OK.

                                                    Note To clear the buffer, select Edit > Clear Buffer.
